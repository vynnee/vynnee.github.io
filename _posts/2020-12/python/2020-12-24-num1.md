**02 서울의 기온 데이터 분석하기**

(1) csv 파일에서 데이터 읽어오기
```java
import csv
f = open('seoul.csv', 'r', encoding='cp949')
data = csv.reader(f, delimiter = ',')
print(data)
f.close()
```
